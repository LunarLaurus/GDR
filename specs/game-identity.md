# Game Identity & Branding

## Topic of Concern
Rebranding the Doom engine from "DOOM" / "Chocolate Doom" to "Goblin Dice Rollaz"

## Requirements
- Project name: "Goblin Dice Rollaz"
- Version: 1.0.0
- Binary names: goblin-doom, goblin-setup, goblin-server
- Menu title: "GOBLIN DICE ROLLAZ"
- All DOOM references replaced with Goblin Dice Rollaz
- Quit message modified to goblin-themed flavor text

## Acceptance Criteria
- [x] CMakeLists.txt has project name "Goblin Dice Rollaz"
- [ ] d_main.c displays "Goblin Dice Rollaz" at startup
- [ ] m_menu.c shows "GOBLIN DICE ROLLAZ" title screen
- [ ] Binary outputs renamed to goblin-*
