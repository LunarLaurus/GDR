# Goblin Dice Rollaz - Public Roadmap

A dice-themed Doom engine game featuring percentile-based combat, dwarf enemies, and roguelike progression.

## Version 1.0 - "The Goblin's Fortune"

### Core Dice Combat System
- [x] Percentile-based damage rolls
- [x] Critical hit system with visual feedback
- [x] Multiple dice weapon types (d4, d6, d8, d10, d12, d20, d100)
- [x] Exploding dice mechanic (optional)
- [x] Advantage/Disadvantage system

### Arsenal
- [x] d6 Blaster - Starting weapon
- [x] d20 Cannon - Heavy hitter
- [x] d12 Heavy Impact - Charged smash
- [x] d4 Throwing Knives - Rapid fire
- [x] d8 Balanced mid-tier
- [x] d10 Ricochet - Bouncing projectiles
- [x] Twin d6 Scatter - Close-range burst
- [x] Arcane d20 Beam - Continuous damage
- [x] Cursed Die - High risk, high reward

### Enemies
- [x] Dwarf Soldiers (multiple variants)
- [x] Dwarf Berserker, Engineer, Defender, Marksman
- [x] Dwarf Miner, Captain, Bombardier, Armored
- [x] Goblin Scout, Sneak, Alchemist, Totemist, Shaman
- [x] Boss: Goblin King (multi-phase)
- [x] Boss: Dwarven War Machine

### Powerups
- [x] Critical Hit Boost - Increased crit chance
- [x] Double Damage - 2x damage multiplier
- [x] Dice Fortune - Guaranteed next-hit crit
- [x] Status effect framework (Burning, Frozen, Stunned, Dice Curse)

### Progression (Optional Mode)
- [x] RPG mode toggle
- [x] Experience and leveling
- [x] Permanent stat upgrades

### Map Themes
- [x] Mine/Cavern theme
- [x] Forge/Lava theme
- [x] Treasure Chamber theme
